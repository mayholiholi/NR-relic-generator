# NR Relic Generator

https://mayholiholi.github.io/nightreign-relic-generator/

ELDEN RING NIGHTREIGN の遺物スクリーンショットから、遺物部分だけを自動で切り抜いて一覧画像を生成するWebツール。

SNSやブログに遺物コレクションを載せたいときに便利。

## 機能

- 複数のスクリーンショットを一括アップロード
- 遺物部分を固定座標で自動切り抜き（手動トリミング不要）
- 列数を指定して整列（デフォルト3列）
- 合成した一覧画像をPNGでダウンロード
- 画像はすべてブラウザ内で処理（サーバーへの送信なし）

## 使い方

### 1. スクリーンショットを用意する

NIGHTREIGN のゲーム内で遺物のステータス画面を開き、スクリーンショットを撮る。

> 切り抜き座標が固定値（x:104, y:245, 620x750px）のため、**UIスケールやウィンドウサイズが標準設定のフルスクリーン**で撮影すること。設定が異なると遺物部分がずれる。

### 2. 画像をアップロードする

「画像を選択」から、用意したスクリーンショットを複数選択する。
選択後、プレビューが表示される。

### 3. 列数を指定する

一覧画像の横並び数を設定する（デフォルト: 3列）。
遺物が10個なら4列で3行、5列で2行、といった具合に自動で折り返す。

### 4. 画像を生成・ダウンロードする

「合成」ボタンをクリック。
生成された一覧画像の下にある「ダウンロード」リンクから `relic-list.png` として保存できる。

## 技術構成

単一HTMLファイル。ビルド不要。

- HTML / CSS / JavaScript
- Canvas API（画像の切り抜き・合成）

## ローカルで動かす

`index.html` をブラウザで直接開くだけで動作する。

ローカルサーバーを使う場合↓

```bash
# Python
python3 -m http.server 8000

# Node.js (npx)
npx serve .
```

## 切り抜き座標について

`index.html` 内の `CROP` オブジェクトで切り抜き領域を定義している。

```js
var CROP = { x: 104, y: 245, w: 620, h: 750 };
```

この値はNIGHTREIGNの遺物ステータス画面（フルスクリーン標準UI）に合わせた固定値。
解像度やUI設定が異なる環境では調整が必要になる場合がある。
